# SoftCloud Archive 软件功能简介

> 面向产品/业务，回答「SoftCloud 是什么、能做什么」。

---

## 项目定位

SoftCloud Archive 是一个类似 **华军软件园** 的云端软件下载站。  
软件的二进制安装包完全存放在网盘中，Cloudflare 只负责元数据与中转跳转。

**核心价值**：元数据在 Cloudflare（D1），二进制在网盘（OpenList），用户通过分享链接下载；成本低、易维护。

---

## 功能模块概览

### 前台（访客）

- 按分类、平台、关键字浏览和搜索软件
- 查看软件详情、版本信息、长描述（Markdown）
- 点击「下载」→ 记录日志 → 302 重定向至网盘分享链接（用户浏览器直接访问网盘）

### 用户系统

- 邮箱注册 / 登录
- 查看和修改个人资料
- 投稿软件（需管理员审核后上线）
- 收藏、评论等已预留表结构，可扩展

### 后台管理

- 软件管理：增删改查、上下架、软删除、OpenList 链接
- 存储后端管理：多网盘抽象、容量视图
- 用户投稿审核：待审核列表、通过/驳回
- 更新源配置：GitHub / HTML 自动检测新版本（架构已预留）

### 多语言与主题

- 中文 / 英文双语言包
- 亮色 / 暗色主题切换

## 目标用户


| 角色   | 典型操作                  |
| ---- | --------------------- |
| 访客   | 浏览、搜索、查看详情、下载         |
| 注册用户 | 登录、修改资料、投稿软件          |
| 管理员  | 软件 CRUD、审核投稿、配置存储与更新源 |


---

## 技术栈（一句话）

**Nuxt 3 + Cloudflare Workers + D1 + OpenList**  
前端部署在 Cloudflare Pages，后端为 Workers REST API，数据存 D1，大文件通过 OpenList 聚合网盘 + WebDAV 管理。支持 GitHub Actions push 一键部署 Workers；Pages 与 OpenList（Render）连接仓库后随 push 自动部署。
