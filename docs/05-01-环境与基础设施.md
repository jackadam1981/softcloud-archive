# 05-01 环境与基础设施

> 阶段 1：搭建开发与部署所需的基础环境。

---

## 目标与范围

- 创建 D1 数据库并执行 schema
- 配置 Workers（Wrangler、D1 绑定、环境变量）
- 启动 Nuxt 3 前端
- 部署 OpenList（用于文件存储与分享链接）

---

## 依赖

无（本阶段为起点）。

---

## 任务清单

- [ ] 安装 Node.js 18+、Wrangler
- [ ] 在 Cloudflare Dashboard 创建 D1 数据库
- [ ] 执行 `db/schema.sql` 初始化表结构
- [ ] 配置 `workers/wrangler.toml`：D1 绑定、`JWT_SECRET`、`ADMIN_TOKEN`、`OPENLIST_BASE_URL`
- [ ] 本地运行 `workers/npm run dev`（默认 http://127.0.0.1:8787）
- [ ] 配置 `frontend/.env` 或 `NUXT_PUBLIC_API_BASE`
- [ ] 本地运行 `frontend/npm run dev`（默认 http://localhost:3000）
- [ ] 部署 OpenList（Render / 1Panel / Docker 等），配置网盘与 WebDAV

---

## 关键文件

| 文件 | 说明 |
|------|------|
| `db/schema.sql` | D1 全量建表脚本 |
| `workers/wrangler.toml` | Workers 配置、D1 绑定、环境变量 |
| `frontend/nuxt.config.ts` | Nuxt 配置，含 `apiBase` |
| `dev-all.ps1` | 一键启动本地 Workers + 前端（Windows） |

---

## 验收标准

- Workers 根路径 `GET /` 返回 `{ "ok": true, "service": "SoftCloud API" }`
- 前端可访问首页，并配置 `NUXT_PUBLIC_API_BASE` 指向 Workers
- OpenList 可访问，WebDAV 已启用
